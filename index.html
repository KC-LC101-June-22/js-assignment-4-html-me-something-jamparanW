Websites I've found interesting or useful

PCPartPicker

PCPartPicker is a comparison shopping website that allows users to compare prices and compatibility of computer components on different retailers online.

Ninite

Ninite is a package management system offering that enables users to automatically install popular applications for their Windows operating system. It enables users to make a selection from a list of applications and bundles the selection into a single installer package.

Craiyon

Craiyon, formerly DALL-E mini, is an AI model that can draw images from any text prompt.

Desmos

Desmos is an advanced graphing calculator implemented as a web application and a mobile application written in JavaScript.

AllTrails

AllTrails is a fitness and travel mobile app used in outdoor recreational activities. The service allows users to access a database of trail maps, which includes crowdsourced reviews and images.
